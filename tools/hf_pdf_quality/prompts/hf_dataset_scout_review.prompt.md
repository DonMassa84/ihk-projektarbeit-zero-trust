# Dataset Scout Review Prompt

Bewerte einen Hugging-Face-Dataset-Kandidaten für ein Zero-Trust-/GitHub-Security-Projekt.

## Bewertungskriterien

1. Zweck
2. Datenformat
3. Lizenzlage
4. Personenbezug
5. Eignung für Anomalieerkennung
6. Eignung für GitHub-Security
7. Eignung für Code-Vulnerability Detection
8. Risiko für IHK-PDF
9. Erlaubte Nutzung: Repo-Ausblick / Evaluation / nicht verwenden
10. Prüfungsfeste Formulierung

## Verboten

- Keine Produktivbehauptungen.
- Keine Leistungskennzahlen erfinden.
